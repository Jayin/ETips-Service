```python
from service.subsystem import SubSystem

u = SubSystem("3112002722", '931127')
u.get_course()
```

### 获取用户的成绩列表

```python
from service.subsystem import SubSystem

u = SubSystem("3112002722", '931127')
u._get_stu_info()
```

**Return**

```json
[
    {
        "score_list": [
             ........
            {
                "remark": "", //备注
                "name": "体育（1）",//名称
                "score": "92",//成绩
                "xuefen": "1",// 学分
                "type": "必修",// 课程类型
                "id": "1500011"// 课程代号
            }
            ......
        ],
        "year": 1  //第几个学年
    },
    .......
]

```
